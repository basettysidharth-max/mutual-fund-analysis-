# Mutual Fund Analytics Dashboard

## Project Overview
This project was developed as part of the Bluestock Capstone Project. The objective is to analyze mutual fund industry data and create an interactive dashboard for performance monitoring and investment insights.

## Objectives
- Analyze mutual fund industry trends
- Track SIP inflows and investor activity
- Compare fund performance across categories
- Visualize key financial metrics

## Tools Used
- Python
- Pandas
- Power BI
- GitHub

## Data Sources
- Fund Master Data
- NAV History
- AUM Data
- SIP Inflow Data
- Investor Transaction Data
- Benchmark Indices

## Dashboard Pages

# 1. Industry Overview
- Total AUM
- Number of Schemes
- SIP Inflows
- Total Folios

# 2. Fund Performance
- Risk vs Return Analysis
- NAV Trends
- Fund Performance Metrics

# 3. Investor Analytics
- Investor Transactions
- Category-wise Analysis
- State-wise Investment Trends

# 4. SIP & Market Trends
- SIP Inflow Growth
- Active SIP Accounts
- New SIP Accounts
- Category Inflow Analysis

# Key Findings
- SIP inflows increased steadily from 2022 to 2025.
- Large fund houses contribute the majority of industry AUM.
- Investor participation continues to grow across categories.
